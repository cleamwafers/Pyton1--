関数の種類

  1. 引数も戻り値もない関数

    2. 決まった仕事をさせたい時。例：def1.py

  2. 引数だけある関数

    3. 違うデータを渡して処理を調整したい時。例：def3.py

  3. 戻り値だけある関数

    4. 結果に変化があるので知りたい時。例：def4.py

  4. 引数も戻り値もある関数

    5. データを渡して計算するときや、実行結果を知りたい時。例：def2.py

import:他人が作ったプログラムを読み込んで利用する時。

モジュール:importで読み込むプログラムファイルのこと。

Pythonの『標準モジュール』

  • 数値計算：mathやrandom(ランダム、乱数)
  • 日付や時刻に使う：datetimeやtimeやcalendar
  • データファイルに使う：csvやjson
  • ユーザーインターフェース：tinter

    • モジュール名を省略する(その1):as

    『import モジュール名』を使って読み込むと、
    『モジュール名.関数名』と指定して使うが、プログラムの中で何度もその関数を実行するような場合、
    長いモジュール名だと読みにくくなってしまうので、『import モジュール名 as 省略名』とすることが出来る。

    モジュール名を省略する(その2):from

    『import モジュール名』を使って読み込むと、
    『モジュール名.関数名』と指定して使うが、関数名の前のモジュール名を書かずに済ませる方法もある。
    それが『from モジュール名 import *』とすることが出来る。

    例：chapter02のタートルグラフィックスのプログラムで使っていた方法。
    本来なら『turtle.shape("turtle")』と書くのですが、『from turtle import *』とすることで、
    『shape("turtle")』と省略できていた。

    簡単なプログラムでは省略できて便利だが、複雑になると同じ関数名がぶつかってエラーになったりする。
